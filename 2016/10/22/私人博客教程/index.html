私人博客教程

发表于 2016-10-22   |   分类于 博客教程   |  

一个人有多大的野心，就会有多大的最作为；

这是我在这个博客的第一篇文章，就是想记录下自己搭建博客的过程，对于想学习的同学有一个参考的东西；
之前由于自己的原因，一直配置失败，本地博客可以打开，但是就是上传不到github服务器中，昨晚才叫舍友帮忙，搞好了。

第一步 工具的下载

总共需要两个工具，git和node.js这个大家自己下载就OK(最新版本)，下面给出下载链接和对应的安装图解；

  1. git
  • 下载链接：https://git-scm.com/download/win 按照PC，选择各自版本；
  • 安装教程：http://jingyan.baidu.com/article/90895e0fb3495f64ed6b0b50.html
  1. node
  • 下载链接：https://nodejs.org/en/
  • 安装教程：http://jingyan.baidu.com/article/b0b63dbfca599a4a483070a5.html

我们根据上面的教程来操作；

第二步 通过node的npm安装各种依赖环境使本地可以浏览

  1. 安装hexo
     npm install -g hexo

这个的意思就是在pc上面安装这个hexo环境，下次使用的时候直接初始化就OK了。

  1. 创建自己的博客文件夹

我创建了blog文件夹，进去之后右键，选择git bash窗口；执行下面操作;

     hexo init
  1. 安装各种依赖

此窗口执行下面代码

     npm install
  1. 本地编译，运行
    继续在此路径下执行下面代码
      hexo g
  hexo server
    执行localhost:4000可以看到如下界面就说明我们本地博客已经创建好了，是不是感觉很简单；

图片01

第三步 申请github，新建一个版本库

接下来就是我们申请自己的版本库了，下面是网址自己注册个，同时安装下面顺序进行版本的创建；

gitHub： http://www.github.com/

  • 注意事项

版本库必须是以github.io结尾的。就好比我的域名就是https://andyahui.github.io/ 这里的库名也必须和自己的用户名一致；
其中Repository name的格式是 ‘用户名’.github.io ；

第四步 同步操作，将本地博客发布到网站

  1. 配置博客
     git config --global user.name Andyahui
 git config --global user.email 1193451014@qq.com
    检查是否连接上；通过下面的语句;
     ssh git@github.com
    如果看到欢迎语句，就说明现在已经连接上了。我们就可以进行推送了；
    ssh git@github.com

  2. 修改配置文件 _config.yml的节点deploy

    deploy:
type: git
repo: https://github.com/Andyahui/Andyahui.github.io.git
branch: master

  3. 若是出现下面的错误，我们需要执行下面的代码；

    not found: git

    执行的代码

      npm install hexo-deployer-git --save

    此时我们就可以进行发布了。

     hexo g
 hexo d

    此时我们就可以在线看自己的博客了。Andyahui

第五步 修改主题

这种简易的博客修改主题很方便，我们可以在百度里面找到很多关于hexo的主题，我们需要将其博客的theme目录拉去下来；
这里只需要三步；

  1. 拉去别人的主题

    git clone https://github.com/iissnan/hexo-theme-next themes/next

  2. 修改自己的主题

    theme: next
  3. 清除，编译，运行；
    hexo clean
hexo g
hexo s
    我们就可以看到具体的内容了。
